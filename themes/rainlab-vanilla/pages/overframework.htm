Goed framework en zeer mooie layout. We hebben dit framework uiteindelijk gebruikt omdat het eerste framework wat issues had.

Een grote issue van onze eerste framework (Symfony) was dat het alleen puur voor PHP was dus hadden we een web server nodig die goed functioneerde. Daarnaast hadden we ook een DataBase nodig waar symfony op draaide wat niet heel erg goed werkte.
